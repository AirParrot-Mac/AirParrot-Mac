## Overview

AirParrot Professional Suite is a complete wireless display platform that serves presenters and organizations with the multi-destination streaming, Reflector integration, and management tools needed for professional wireless display workflows.

Multi-destination streaming sends content to multiple receivers simultaneously, enabling display on multiple screens in large venues, classroom environments, and conference settings. Each destination receives the same content with independent quality settings optimized for each receiver's capability and network connection.

Reflector integration works with Squirrels' Reflector software to create wireless display infrastructure where any computer running Reflector becomes a receiver. This combination enables peer-to-peer wireless display between computers without dedicated hardware. The Reflector ecosystem handles classroom and conference scenarios where multiple presenters share to a central display.

Custom streaming presets save resolution, frame rate, and quality configurations for different venues and use cases. Presets handle the varying network conditions and display capabilities that presenters encounter across different locations. Quick switching between presets adapts streaming quality without manual reconfiguration.

AirParrot Professional Suite is the wireless display platform for professionals who need flexible, cross-ecosystem screen mirroring and streaming with the multi-destination capability and quality control that professional presentations and collaboration require.
